Rezi Bullet Point SDK

AI-powered resume bullet point generator. Enter a job title, select skills, and generate tailored bullet points instantly.

Quick Start

Add this snippet to your HTML. No CSS file needed — styles are bundled in the script.

<!-- 1. Place the container where you want the widget --> <div data-bullet-generator data-recaptcha-key="YOUR_RECAPTCHA_SITE_KEY" data-cta-url="https://rezi.ai/signup" data-max-skills="5" ></div> <!-- 2. Load the SDK script --> <script src="https://cdn.jsdelivr.net/npm/@rezi/bulletpoint-sdk/rezi-bullets.min.js"></script>

Data Attributes

AttributeDescriptionDefault
data-bullet-generator Marks the element as the SDK container. Required -
data-recaptcha-key reCAPTCHA Enterprise site key for API access. Required -
data-cta-url URL to redirect when CTA button is clicked or rate limit is reached. -
data-max-skills Maximum number of skills a user can select. 5
data-dev Enable dev mode (console logs). Remove for production. false

Manual Initialization (with callbacks)

For more control, initialize the SDK via JavaScript instead of data attributes.

<div id="my-container"></div> <script src="https://cdn.jsdelivr.net/npm/@rezi/bulletpoint-sdk/rezi-bullets.min.js"></script> <script> const generator = new ReziBulletSDK.BulletPointGenerator({ containerSelector: '#my-container', ctaUrl: 'https://rezi.ai/signup', maxSkillSelection: 5, // Called when bullet points are generated onGenerate: (bullets) => console.log('Generated:', bullets), // Called when a bullet is copied onCopy: (bullet) => console.log('Copied:', bullet), // Called when CTA button is clicked (before redirect) onCTAClick: (bullets, jobTitle) => { localStorage.setItem('resume_prefill', JSON.stringify({ jobTitle, bullets })); }, }); </script>
